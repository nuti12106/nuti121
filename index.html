Gundam
ยุทโธปกรณ์ 60mm Vulcan Gun X 2 Beam Magnum X 1 Beam Saber X 4 Shield (I-Field Generator) X 1 นักบิน Banagher Links ดีไซเนอร์ Hajime Katoki ส่วนสูง 19.7 เมตร (Unicorn Mode) 21.7 เมตร (Destroy Mode) น้ำหนัก 23.7 T RX-0 UNICORN GUNDAM ผู้ผลิต Anaheim Electronics ประวัติ
ผลิตขึ้นในช่วงสุดท้ายของของ แผนการปรับโครงสร้างกองกำลังโลก E.F.S.F (Earth Federation Space Force) ในโครงการ "UC Project" ผลผลิตของโครงการมีทั้งหมดสามเครื่องได้แก่ "RX-0 UNICORN GUNDAM" "RX-0 UNICORN GUNDAM 02 BANSHEE" "RX-0 UNICORN GUNDAM 03 PHENEX" จุดประสงค์ที่แท้จริงของโครงการคือการกาดล้าง Newtype ให้หมดสิ้น โดยสองเครื่องแรกได้ถูกสร้างขึ้นที่โรงงานของAnaheim Electronics และเครื่องที่สามได้ถูกสร้่งขึ้นอย่างเป็นความลับที่โรงงานของ E.F.S.F (Earth Federation Space Force) อย่างไรก็ตามมูลนิธิวิสท์ได้เข้าควบคุมโครงการ แล้วทำการยืม "RX-0 UNICORN GUNDAM" ไป พวกเขาได้ทำการติดตั้งโปรแกรม Laplace Program เพื่อค้นหา Newtype ที่แท้จริงทำให้จุดประสงค์ดั้งเดิมเปลี่ยนไป ด้วยเทคโนโลยีโมบิลสูทที่ล้ำหน้าที่สุดในยุคนั้น ประสิทธิภาพของมนจึงไม่มีใครเทียบได้ โดยสร้างจากการรวมรวบข้อมูลจาก MSN-06S SINANJUN รุ่นทดลองที่ถูกขโมยไปโดยนีโอซีออน ทำให้ Unicorn สามารถติดตั้ง Psycho-frame ได้ทั้งตัว ซึ้งช่วยให้นักบินสามารถควนคุมได้ง่ายมากขึ้น แต่ก้จะให้นักบินต้องรับภาระทางสมองมากขึ้นอีกด้วยทำให้เวลาการทำงานถูกจำกัดอยู่ที่ 3 นาที
